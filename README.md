# 📊 Indian Stock Market Analytics Dashboard

## 📌 Project Overview

A **comprehensive analytics platform** designed specifically for the **Indian Capital Markets**. This dashboard analyzes stock performance from the **National Stock Exchange (NSE)** and **Bombay Stock Exchange (BSE)**, providing:

- 🔥 **Real-time Technical Analysis** (RSI, MACD, Bollinger Bands, Moving Averages)
- 📉 **Candlestick Charting** with volume profiles  
- 🏦 **Sector-wise Performance** (Banking, IT, Pharma, Auto, Energy)
- 💼 **Portfolio Tracking** & Backtesting strategies
- 🌡️ **Market Sentiment Analysis** (India VIX, FII/DII flows concept)

> **Target Audience:** Retail investors, algo-traders, data analysts studying Nifty/Sensex trends.

## ✨ Key Features & Capabilities

### 🎯 Core Modules

<details>
<summary><b>1️⃣ Market Overview (Macro View)</b></summary>

- **Indices Tracking**: Nifty 50, Nifty Bank, Sensex, India VIX
- **Sector Heat Map**: Color-coded grid of sectoral indices (Nifty IT, Pharma, PSU Bank)
- **FII/DII Activity**: Net buying/selling trends *(if data available)*
- **Market Breadth**: Advance-Decline ratio for overall sentiment

</details>

<details>
<summary><b>2️⃣ Individual Stock Screener</b></summary>

Search any **NSE/BSE symbol**:
- `RELIANCE`, `TCS`, `INFY`, `HDFCBANK`, `ICICIBANK`, `SBIN`, `WIPRO`, `LT`
- **OHLCV Visualization**: Open, High, Low, Close, Volume over custom periods
- **Price Action**: Support/Resistance levels identified automatically
- **52-Week High/Low**: Distance from extreme points in %

</details>

<details>
<summary><b>3️⃣ Technical Indicators Engine</b></summary>

Calculate and visualize:
- **Trend Following**: SMA(50), EMA(200), VWAP, Parabolic SAR
- **Momentum**: RSI(14), Stochastic Oscillator, ROC
- **Volatility**: Bollinger Bands (20,2), ATR, Historical Volatility
- **Moving Averages**: Golden Cross / Death Cross detection on Indian blue chips

</details>

<details>
<summary><b>4️⃣ Fundamental Metrics*</b></summary>

*(If using screener APIs like Moneycontrol/Tijori)*
- P/E Ratio, P/B Ratio comparison vs sector average
- Dividend Yield history
- EPS Growth trends (quarterly)
- Debt-to-Equity ratios for top Indian companies

</details>
